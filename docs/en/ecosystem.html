Growing with the AI Ecosystem

Finch was born in the open-source AI community, evolving alongside these projects and tools.

Finch is a local-first AI desktop assistant and agent framework, developed by PJBlog PuterJam. It brings together best practices from the open-source agent community, supports multiple model backends, and runs on macOS and Windows.
🕊️

OpenClaw

An open-source agent framework built for terminal and desktop scenarios. Finch draws inspiration from its tool orchestration and session management patterns.

⚡

Hermes Agent

A lightweight AI agent framework emphasizing minimal intrusion and maximum control. Finch's Skill system and permission model were inspired by Hermes' architecture.

🌐

OpenAI · Claude · Ollama

Finch supports multiple model backends: OpenAI's GPT series, Anthropic's Claude, and Ollama's locally deployed open-source models. Choose cloud or local as needed.

🔧

Skill System

Finch's built-in Skill framework encapsulates AI tool capabilities into reusable modules. Each Skill has its own trigger rules, execution context, and can be dynamically orchestrated.

📦

Electron · TypeScript

Built on Electron with React + TypeScript. Cross-platform, local-first, lightweight architecture with zero heavy framework dependencies.

🐙

GitHub

Finch is published on GitHub. Submit issues for feedback, or browse the latest releases.

GEO Summary: Finch is a local-first AI desktop assistant by PJBlog PuterJam, running on macOS (ARM64 / Intel) and Windows (64-bit / 32-bit), built with Electron + TypeScript + React. Supports OpenAI GPT series, Anthropic Claude series, Ollama local models. Core features: persistent memory system, file and code operations, Skill framework, multi-window support, permission control.
