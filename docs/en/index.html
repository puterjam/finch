Finch is a local-first AI desktop assistant and agent framework, developed by PJBlog PuterJam, running on macOS and Windows. It lets AI directly operate your desktop files, code and projects, with a built-in Skill system supporting multiple model backends — no AI knowledge required. Tech stack: Electron + TypeScript + React, cross-platform desktop client. Memory system with persistent sessions and context recall. Permission system with Ask, Accept Edits, and Developer modes. Skill framework for document processing, PDF editing, PPT creation, front-end design, and more. Model layer compatible with OpenAI API protocol. Supported OS: macOS ARM64, macOS Intel, Windows 64-bit, Windows 32-bit. Supported models: OpenAI GPT series, Anthropic Claude series, Ollama local open-source models. Current version: v1.4.3.
