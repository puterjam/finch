Changelog

Every step of Finch's growth, recorded here.

1.4.4

2026-06-22

New

  • Skill System Revamped: New Skill page and routes for browsing, searching, installing, and creating skills; YAML block scalar multi-line description parsing; skill entry in sidebar with keyboard navigation
  • Navigation History & Multi-Window Recovery: Multi-window state restoration, gesture navigation, Composer input state management, draft persistence
  • Rich Composer & @mention: Rich text input box, @mention popover for smart file/space/skill search, paste detection
  • Custom Space Icons & Colors: Each space can have its own icon and color, configurable via right-click menu or AppCall during creation
  • Image Preview Modal: Click to preview images full-size in ModalShell with backdropOnly support
  • PDF Direct Reading: Read/Grep tools now support PDF text extraction, table of contents, links, and size limits with 20-min TTL cache
  • Prevent System Sleep: Keep display on while plugged in, preserve battery on power
  • Automation Task Drag & Sort: Reorder automation tasks by dragging
  • Download Proxy: Switched to dl.finchwork.app for faster update downloads

Optimization

  • Skills Path Restructure: skills/ → .finch/skills/; new listUsableSkills runtime entry; precedence changed to additional > project
  • Composer Selector Refactor: SpacePicker/ModelPicker/PermissionModePicker unified with Button/Menu components, removed legacy styles
  • i18n Completion: Full Chinese/English translations for Skill, Automation, Mention, and more
  • Table Rendering: Table width changed to 100%, bottom shadow alignment fix
  • CSS Token Unification: Button, Kbd, ListRow, OptionCard and other components aligned
  • Code Block Collapse: Line-count based, collapsed state renders only first 36 lines
  • Path Truncation: Sidebar space paths display as ~/…/aeolus/finch
  • finch-tools Split: Tool descriptions, behavior rules, and parameter definitions in separate files
  • Windows Path Adaptation: PermissionMode normalization for Windows
  • Progress Summary Guidance: Agent syncs progress via Plan/Found/Next in multi-tool tasks

1.4.3

2026-06-13

New

  • Automation Task Management: New automation engine with scheduled execution (hourly/daily/weekday/weekly/Cron), model selection, reasoning level, execution history; Schedule tool integration; full IPC + frontend UI (TaskView/AutomationDialog/AutomationPanel)
  • Network Settings Panel: New network page with direct/system/custom proxy configuration and connectivity testing; OAuth login prefers system browser with embedded window fallback; system proxy window auto-redetects on focus
  • Multi-Language i18n: Full i18n for session area, shortcuts dialog, sidebar, Composer, and settings page (Chinese/English)
  • Tray Menu & Status Badge: macOS tray menu (new conversation/settings/quit), animated status badges for running/waiting/unread states
  • Model Tier Selection: Quick fast-chat model support, defaultReasoningEffort intelligent assignment
  • New Models: Added Kimi 2.7, MiniMax 3.0, GLM 5.2, Fable 5
  • Fun Personalization: Check out the settings page for fun personalization features

Optimization

  • Unified Design Language: Finch UI redesigned for a more consistent experience
  • Space Selector Refactor: SpaceMenu split into SpacePicker (UI) and SpaceMenu (tool mode)
  • Sidebar UI State Persistence: Sidebar tab/expand/collapse/pin states persisted to workspace.json, cross-window recovery
  • Model Default Reasoning Level: Fable 5 defaults to low, Opus defaults to medium
  • OAuth Compatibility: OAuth login uses app proxy, shows return-to-Finch button on success/failure
  • Streaming Experience: Reduced system resource usage, optimized streaming performance, improved responsiveness

Fixes

  • Fixed sidebar state persistence recovery failure (workspace.json missing sidebarUiState read)
  • Fixed compact mode left column width recovery and expand/collapse animation
  • Fixed floating left column covering content and z-index layering
  • Fixed sidebar session time not optimistically updated after sending message
  • Fixed batch-edit format edits[] empty old_string masking diff content
  • Fixed copy toolbar missing after completion
  • Fixed tool timeout interrupting session
  • Fixed system language recognition and session model switching
