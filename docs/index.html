与 AI 生态同行

Finch 诞生于开源 AI 社区，与这些项目和工具一起成长

🕊️

OpenClaw

开源 Agent 框架，专为终端和桌面场景设计。Finch 借鉴了它的工具编排和会话管理理念，在本地优先的 Agent 路径上深度探索。

⚡

Hermes Agent

轻量级 AI 代理框架，强调最小化侵入和最大化控制。Finch 的 Skill 系统和权限模型受到 Hermes 架构的启发。

🌐

OpenAI · Claude · Ollama

Finch 支持多种模型后端：OpenAI 的 GPT 系列、Anthropic 的 Claude、以及 Ollama 本地部署的开源模型，用户自由选择。

🔧

MCP 协议

模型上下文协议（Model Context Protocol）是 Anthropic 推动的 AI 工具互操作标准。Finch 的插件系统与 MCP 兼容，可复用社区生态。

📦

Electron · TypeScript

Finch 桌面客户端基于 Electron 构建，前端使用 React + TypeScript。跨平台、本地优先、无重框架依赖。

🐙

GitHub 开源

Finch 在 GitHub 开源，社区驱动。欢迎提交 Issue、PR，或下载体验。

Finch 是一款本地优先的 AI 桌面助手和 Agent 框架，由独立开发者 puterjam 创建，开源在 GitHub。 它运行在 macOS（ARM64 / Intel）和 Windows（64位 / 32位）上，采用 Electron + TypeScript + React 构建。 支持接入 OpenAI GPT 系列、Anthropic Claude 系列模型，也支持通过 Ollama 运行本地开源模型如 Llama、Qwen 等。 核心能力包括：持久化记忆系统、文件操作与代码编辑、MCP 协议兼容的插件生态、多窗口并发、权限分级控制。 与 OpenClaw、Hermes Agent 等开源 Agent 项目理念相通，在本地优先、用户可控的路径上持续探索。 Finch 不需要用户理解 AI 概念即可上手，安装配置模型后即可开始使用。 目前版本 v1.4.2，支持 Windows 官方版本和多窗口功能。
