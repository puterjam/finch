更新日志

Finch 的每一次成长，都在这里记录。

1.2.0

2026-05-16

新增

  • 在线自动更新：接入 electron-updater，启动后静默检查 GitHub Release，发现新版本后台自动下载，用户决定何时重启安装
  • 首页更新提示气泡：首页右上角实时显示更新状态（发现新版本 / 下载进度 / 点击重启安装），品牌绿配色
  • 预发布通道：设置 → 关于 新增"接收预发布版本"开关，可提前体验 beta 版本
  • 身份文件快照：重新孵化或覆盖前，身份文件自动备份至 ~/.finch/snapshots/，保留最近 20 份
  • 智谱 GLM Code Plan：新增内置供应商，支持 GLM-5.1 / GLM-4.7 / GLM-4.5-Air，200k 上下文
  • 工作区面板初始扫描：新建会话时自动扫描当前目录并展示文件结构，不再显示空白面板
  • 工具调用动画：运行中的工具调用显示限流滚动动画，超长路径自动截断美化展示

优化

  • 启动速度：冷启动白屏时间大幅缩短，webview 提前预加载，打开时不再闪白屏
  • 包体积：PNG 资源全量压缩，安装包体积减少约 81%
  • Context Window 统计：改用 assistant 消息级别 usage 计算，避免数值虚高
  • 更新状态管理：更新逻辑抽离为独立 updateStore，后台每 4 小时自动检查，错误提示中文化
  • 会话列表排序：过滤 SDK metadata 行 mtime 干扰，排序结果稳定不跳动

Bug 修复

  • backdrop-filter 失效：修复打包后 blur 特效失效的问题（browserslist 配置导致）
  • AskUserQuestion 选项截断：移除 description 副标题展示，label 保证完整可读
  • 工作区面板：修复新建会话后面板显示为空、滚动区域、分组展示三处异常
  • 重新孵化后引导语：修复重新孵化后首页未自动填入引导语的问题

1.1.0

2026-05-15

新功能

  • Mermaid 图表渲染：CodeBlock 支持 Mermaid 图表实时渲染，可在预览与源码间切换
  • HTML 文件预览：HTML 文件改为 webview 渲染，支持真实页面预览而非展示源码
  • 附件持久化：附件文件现可跨会话持久化，会话视图同步改造
  • 工作目录侧栏折叠：WorkSidebar 工作目录分区支持折叠并记忆折叠状态
  • Markdown 预览：CodeBlock 支持 Markdown 预览与原始内容切换
  • HATCH 孵化流程：重写 HATCH 孵化流程，与 user-profile 完整联动

性能优化

  • 消除启动白屏：优化启动性能，消除应用启动时白屏闪烁
  • webview 预加载：webview 在动画开始前提前加载，消除打开时的白屏

界面优化

  • 外观设置重设计：新增字体大小调节、主题分段控件，主目录卡片全面重设计
  • 设置窗口宽度限制：打开设置时强制窗口宽度不低于 960px，关闭后恢复限制
  • 设置页装饰图：设置页装饰图随窗口宽度自适应缩放
  • 首页提示文案：优化首页提示文案样式与目录按钮展示
  • 模型设置页面：优化模型设置页面交互细节
  • 侧栏交互：优化侧栏交互体验

Bug 修复

  • 历史会话继续对话：修复搬家后历史会话无法继续对话的问题
  • 推理底部指示器：修复推理中底部指示器抖动问题
  • 推理历史渲染：区分推理中断的历史渲染状态
  • AskUserQuestion 文本长度：限制问题文本长度，避免提问过长
  • 设置关闭通知：改用 useEffect 清理函数触发 notifySettingsClose，修复时序问题
  • 会话输入与推理展示：优化会话输入与推理代码块展示

其他

  • 安全依赖升级：升级 Electron，修复 18 个 Dependabot 安全告警
  • CN 服务商补充：补充各 CN 服务商 apiKeyUrl 并修正模型上下文长度
