更新日志

Finch 的每一次成长，都在这里记录。

1.4.3

2026-06-13

新增

  • 自动化任务管理：全新自动化引擎，支持定时执行（小时/日/工作日/周/Cron）、模型选择、推理等级、执行历史记录；Schedule 工具集成；全链路 IPC + 前端 UI（TaskView/AutomationDialog/AutomationPanel）
  • 网络设置面板：新增网络页，支持直连/系统代理/自定义代理配置与连通性测试；OAuth 登录优先系统浏览器，回退内嵌窗口；系统代理窗口获焦自动重检测
  • 多语言国际化：会话区域、快捷鍵对话框、侧栏、Composer、设置页全面 i18n 中英文翻译
  • 托盘菜单 & 状态角标：macOS 托盘菜单（新对话/设置/退出），运行/等待/未读状态动画角标
  • 模型分级选择：支持 Quick 快速对话模型，defaultReasoningEffort 智能分配推理等级
  • 新增模型：新增 Kimi 2.7, MiniMax 3.0, GLM 5.2, Fable 5
  • 好玩的个性化功能：到设置界面里看看吧

优化

  • 统一设计语言重构：Finch UI 重新设计，体验更一致了
  • 空间选择器重构：SpaceMenu 拆分为 SpacePicker（UI）和 SpaceMenu（工具模式）
  • 侧栏 UI 状态持久化：侧栏 tab/展开/收起/钉住等状态统一持久化到 workspace.json，支持跨窗口恢复
  • 模型配置默认推理等级：Fable 5 默认 low，Opus 默认 medium，更合理默认值
  • OAuth 兼容性：OAuth 登录走应用代理，成功/失败均显示返回 Finch 按钮
  • 推流体验优化：减少系统资源占用，优化推流性能，提升响应速度

修复

  • 修复 sideBar 状态持久化恢复失效（workspace.json 漏读 sidebarUiState）
  • 修复紧凑模式左栏宽度恢复与展开收起动画
  • 修复悬浮左栏遮盖内容与 z-index 层级
  • 修复发送消息后侧栏会话时间未乐观更新
  • 修复 batch-edit 格式下 edits[] 空 old_string 遮蔽 diff 内容
  • 修复结束后复制工具条缺失
  • 修复工具超时中断会话
  • 修复系统语言识别与会话模型切换

1.4.1

2026-06-01

新增

  • Claude & OpenAI 模型全面接入：支持 Claude 订阅、Anthropic API 及 OpenAI API，在设置中选择接入方式即可切换使用，无需重启
  • OAuth 登录：支持通过 OAuth 授权登录云端服务，不再需要手动填写 API Key
  • 对话分支：在任意 AI 回复上点击分支按钮，从当前上下文开启新方向继续对话，原会话保持不变
  • 空间与目录关联：空间可绑定本地目录，AI 自动感知工作区上下文；目录空间与普通空间区分展示
  • 后台任务管理：AI 可在后台执行长时间任务，完成后推送通知；关闭主窗口后任务照常运行
  • 只读命令自动放行：ls、cat、grep 等只读操作不再弹出权限确认
  • 附件支持升级：图片自动压缩并显示缩略图，PDF 支持本地文本抽取后直接传给 AI
  • WebFetch / WebSearch：支持网页抓取与搜索，AI 可主动获取最新信息辅助回答
  • ⌘⇧B 快捷键：随时快速展开或收起右侧工具栏
  • 支持后端推理：客户端窗口关闭不会打断模型的长任务工作

优化

  • 记忆能力提升：会话结束后自动提炼关键信息并写入记忆，跨会话保持上下文；记忆搜索按日期降序优先展示最近内容
  • 工作抽屉交互重做：工具调用、文件变更、命令执行按类型聚合展示，任务完成后只保留文件数、行差、耗时等有效信息
  • 对话列表可读性提升：连续 AI 回复自动合并为一条展示；隐藏 Thinking 时相邻回复同步合并
  • 权限与提问卡片全面重构：统一卡片管理系统，窗口切换后不再消失；响应后立即关闭；拒绝权限时批量关闭
  • 会话状态指示器重做：运行中由呼吸圆点升级为旋转加载环，等待确认时显示脉冲发光
  • 运行中实时显示 token 用量：对话进行中即可看到本次 turn 的 token 消耗
  • 批量归档会话：标题栏菜单支持一键归档 7 天不活跃的会话
  • Thinking 块视觉优化：推理进行时自动滚底跟踪；去掉背景 grain 纹理
  • 内存性能优化：会话 timeline 启用 LRU 淘汰策略，后台轮询请求合并
  • Bash 环境修复：命令执行改为加载 shell profile，解决 Electron 启动时 PATH 缺失问题
  • 上下文超长错误引导：显式提示内容过长时可开启新会话或分多次发送，错误旁增加「开启新会话」按钮

修复

  • 连续发送消息不再丢失：修复快速连续发送多条消息时的队列竞态问题
  • 窗口切换后不再叠字：切到后台再切回，AI 回复内容不再出现重复渲染
  • 权限卡与提问卡窗口重开后可用：重新打开窗口后未完成的权限请求和提问卡片正常恢复
  • 编辑消息后对话正常延续：修复编辑历史消息后分支数据错误，编辑后的对话完整保留上下文
  • 侧边栏幽灵会话消失：修复 session 扫描将普通消息 ID 误注册为 session 的问题
  • Thinking 推理完成后折叠状态正确：修复推理结束后 thinking 块意外全部折叠的问题
  • 工具超时不再误中断等待：等待用户确认权限或提问期间暂停超时计时，响应后重启
  • 背景会话完成状态同步：修复后台运行的会话完成后侧边栏状态不更新的问题
  • 新建会话不误清空间选择：修复新建 session 时错误重置当前已选工作空间的问题
  • 会话归档/删除日期解析修复：修复迁移会话时日期字段存储为无法解析的文本的问题
