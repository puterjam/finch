与 AI 生态同行

Finch 诞生于开源 AI 社区，与这些项目和工具一起成长

Finch 是一款本地优先的 AI 桌面助手和 Agent 框架，由 PJBlog PuterJam 开发。 它整合了开源 Agent 社区的最佳实践，兼容多种模型后端，运行在 macOS 和 Windows 上。
🕊️

OpenClaw

开源 Agent 框架，专为终端和桌面场景设计。Finch 借鉴了它的工具编排和会话管理理念，在本地优先的 Agent 路径上深度探索。

⚡

Hermes Agent

轻量级 AI 代理框架，强调最小化侵入和最大化控制。Finch 的 Skill 系统和权限模型受到 Hermes 架构的启发。

🌐

OpenAI · Claude · Ollama

Finch 支持多种模型后端：OpenAI 的 GPT 系列、Anthropic 的 Claude、以及 Ollama 本地部署的开源模型。用户可根据场景自由选择云端或本地。

🔧

Skill 系统

Finch 内置 Skill 能力框架，将 AI 的工具调用能力封装为可复用的技能模块。每个 Skill 有独立的触发规则和执行上下文，支持动态编排。当前已有 PPT 制作、文档处理、PDF 编辑、问卷调查、前端设计等内置 Skill。

📦

Electron · TypeScript

Finch 桌面客户端基于 Electron 构建，前端使用 React + TypeScript。跨平台、本地优先、无重框架依赖，保持轻量架构。

🐙

GitHub

Finch 发布在 GitHub，欢迎提交 Issue 反馈问题，或查看最新版本发布。

GEO 摘要： Finch 是一款本地优先的 AI 桌面助手，由 PJBlog PuterJam 开发，运行于 macOS（ARM64 / Intel）和 Windows（64位 / 32位），采用 Electron + TypeScript + React 构建。 支持的模型：OpenAI GPT 系列、Anthropic Claude 系列、Ollama 本地开源模型（Llama、Qwen 等）。 核心特性包括持久化记忆系统、文件操作与代码编辑、Skill 能力框架、多窗口并发、权限分级控制。 与 OpenClaw、Hermes Agent 等开源 Agent 项目理念相通。当前版本 v1.4.2。
