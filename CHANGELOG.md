# Changelog

## [1.1.0] - 2026-05-15

### 新功能

- **Mermaid 图表渲染**：CodeBlock 支持 Mermaid 图表实时渲染，可在预览与源码间切换；推理内容中内嵌 Mermaid 不再触发渲染
- **HTML 文件预览**：HTML 文件改为 webview 渲染，支持真实页面预览而非展示源码
- **附件持久化**：附件文件现可跨会话持久化，会话视图同步改造
- **工作目录侧栏折叠**：WorkSidebar 工作目录分区支持折叠并记忆折叠状态
- **Markdown 预览**：CodeBlock 支持 Markdown 预览与原始内容切换
- **HATCH 孵化流程**：重写 HATCH 孵化流程，与 user-profile 完整联动
- **checkPathsExist 接口**：desktopApi 补充 `checkPathsExist` 方法桩

### 性能优化

- **消除启动白屏**：优化启动性能，消除应用启动时白屏闪烁
- **webview 预加载**：webview 在动画开始前提前加载，消除打开时的白屏

### 界面优化

- **外观设置重设计**：新增字体大小调节、主题分段控件，主目录卡片全面重设计
- **设置窗口宽度限制**：打开设置时强制窗口宽度不低于 960px，关闭后恢复限制
- **设置页装饰图**：设置页装饰图随窗口宽度自适应缩放
- **首页提示文案**：优化首页提示文案样式与目录按钮展示
- **模型设置页面**：优化模型设置页面交互细节
- **主工作目录卡片**：重设计主工作目录设置卡片样式
- **侧栏交互**：优化侧栏交互体验

### Bug 修复

- **历史会话继续对话**：修复搬家后历史会话无法继续对话的问题
- **推理底部指示器**：修复推理中底部指示器抖动问题
- **推理历史渲染**：区分推理中断的历史渲染状态
- **AskUserQuestion 文本长度**：限制 AskUserQuestion 问题文本长度，避免提问过长
- **设置关闭通知**：改用 `useEffect` 清理函数触发 `notifySettingsClose`，修复时序问题
- **会话输入与推理展示**：优化会话输入与推理代码块展示

### 重构

- **通用组件收敛**：将通用组件收敛到 `ui` 层，统一复用层级
- **设置组件复用**：优化设置组件复用与模型配置交互
- **前端组件梳理**：梳理前端组件结构，修复运行态交互问题

### 其他

- **安全依赖升级**：升级 Electron 与 Claude Agent SDK，修复 18 个 Dependabot 安全告警
- **CN 服务商补充**：补充各 CN 服务商 `apiKeyUrl` 并修正模型上下文长度
- **运行时数据文档**：记录运行时数据和 session 存储位置在 `~/.finch`

## [1.0.0] - 2026-05-14 (首次发布)

### 核心功能

- Electron + React + TypeScript 桌面应用
- Claude Agent SDK 集成，支持多轮对话
- 会话创建、历史记录与恢复
- 工作目录管理与多目录切换
- 模型配置，支持多家 AI 服务商
- 设置弹窗与 IPC 通信架构
- macOS DMG 安装包，支持 Retina 背景图与自动版本号同步
