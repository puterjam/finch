# Changelog

## [1.4.1] - 2026-06-01

### 新增

- **Claude & OpenAI 模型全面接入**：支持 Claude 订阅、Anthropic API 及 OpenAI API，在设置中选择接入方式即可切换使用，无需重启。
- **OAuth 登录**：支持通过 OAuth 授权登录云端服务，不再需要手动填写 API Key。
- **对话分支**：在任意 AI 回复上点击分支按钮，从当前上下文开启新方向继续对话，原会话保持不变。
- **空间与目录关联**：空间可绑定本地目录，AI 自动感知工作区上下文；目录空间与普通空间区分展示，避免重复关联。
- **后台任务管理**：AI 可在后台执行长时间任务，完成后推送通知；关闭主窗口后任务照常运行。
- **只读命令自动放行**：`ls`、`cat`、`grep` 等只读操作不再弹出权限确认，减少不必要的打断。
- **附件支持升级**：图片自动压缩并显示缩略图，PDF 支持本地文本抽取后直接传给 AI。
- **WebFetch / WebSearch**：支持网页抓取与搜索，AI 可主动获取最新信息辅助回答。
- **⌘⇧B 快捷键**：随时快速展开或收起右侧工具栏。
- **支持后端推理**：客户端窗口关闭不会打断模型的长任务工作。

### 优化

- **记忆能力提升**：会话结束后自动提炼关键信息并写入记忆，跨会话保持上下文；记忆搜索按日期降序优先展示最近内容。
- **工作抽屉交互重做**：工具调用、文件变更、命令执行按类型聚合展示，不同类别使用专属图标；任务完成后只保留文件数、行差、耗时等有效信息，信息密度大幅降低。
- **对话列表可读性提升**：连续 AI 回复自动合并为一条展示，去除中间多余的工具条；隐藏 Thinking 时相邻回复同步合并，界面更清爽。
- **权限与提问卡片全面重构**：统一卡片管理系统，窗口切换后不再消失；响应后立即关闭，不等整轮完成；拒绝权限时同轮所有卡片批量关闭。
- **会话状态指示器重做**：运行中由呼吸圆点升级为旋转加载环，等待确认时显示脉冲发光；后台会话完成状态实时同步到侧边栏。
- **运行中实时显示 token 用量**：对话进行中即可看到本次 turn 的 token 消耗，不再等回复结束才更新。
- **批量归档会话**：标题栏菜单支持一键归档 7 天不活跃的会话，操作前显示确认弹窗防止误触。
- **Thinking 块视觉优化**：推理进行时自动滚底跟踪；去掉界面背景 grain 纹理，视觉更简洁。
- **内存性能优化**：会话 timeline 启用 LRU 淘汰策略，后台轮询请求合并，大量会话场景下内存占用明显降低。
- **Bash 环境修复**：命令执行改为加载 shell profile，解决 Electron 启动时 PATH 缺失导致常用工具找不到的问题。
- **上下文超长错误引导**：显式提示内容过长时可开启新会话或分多次发送，错误旁增加「开启新会话」按钮。

### 修复

- **连续发送消息不再丢失**：修复快速连续发送多条消息时的队列竞态问题，所有消息均可按序送达并处理。
- **窗口切换后不再叠字**：切到后台再切回，AI 回复内容不再出现重复渲染。
- **权限卡与提问卡窗口重开后可用**：重新打开窗口后未完成的权限请求和提问卡片正常恢复，不再无法回答。
- **编辑消息后对话正常延续**：修复编辑历史消息后分支数据错误，编辑后的对话完整保留上下文并正常继续。
- **侧边栏幽灵会话消失**：修复 session 扫描将普通消息 ID 误注册为 session，导致侧边栏出现无标题空白会话的问题。
- **Thinking 推理完成后折叠状态正确**：修复推理结束后 thinking 块意外全部折叠的问题。
- **工具超时不再误中断等待**：等待用户确认权限或提问期间暂停超时计时，响应后重启，不再因等待时间过长被强制中断。
- **背景会话完成状态同步**：修复后台运行的会话完成后侧边栏状态不更新、持续显示运行中的问题。
- **新建会话不误清空间选择**：修复新建 session 时错误重置当前已选工作空间的问题。
- **会话归档/删除日期解析修复**：修复迁移会话时日期字段存储为无法解析的文本，导致部分旧会话无法删除或归档的问题。

## [1.3.0] - 2026-05-21

### 新功能

- **完整记忆系统（测试版）**：会话结束时自动蒸馏对话为事实，分类写入身份与记忆文件，支持信任评分与低分归档
- **MCP 应用桥接**：通过 AI 交互管理空间（罗列/创建/重名/删除）、调整外观设置、查询版本与执行应用升级
- **空间路由与工作目录**：空间可关联本地目录，目录空间与普通空间区分展示
- **会话自动命名**：AI 根据对话内容自动为无标题会话生成名称
- **对话压缩状态提示**：上下文触发自动压缩时显示压缩中动画与完成标记
- **设置页增强**：新增辅助模型下拉选择、记忆模式配置与统计面板
- **工作区交互优化**：文件名超长截断、规则文件图标与点击定位
- **右侧工具栏重构**：支持 docked / floating 双模式

### 性能优化

- **渲染性能大幅提升**：滚动容器、静态消息、代码块全面启用合成层优化
- **SyntaxHighlighter 懒加载**：流式输出跳过高亮，完成后渲染
- **推理实时统计**：cache 和输出 token 在推理过程中实时更新

### 优化

- **记忆搜索改进**：搜索日志文件而非已注入内容，结果按日期排序
- **侧栏视觉统一**：分区标题置顶，修复缩窄时换行与 icon 消失
- **更新通知重做**：全局固定显示，进度条与按钮更醒目

### Bug 修复

- 修复切换会话时 Composer 模型选择不更新
- 修复键盘导航跳过空间会话
- 修复 Composer 输入框在页面挤压时不增高
- 过滤 task-notification 防止内部事件泄漏到界面
- 修复空间目录路径未规范化导致去重失败
- 修复多 provider token 计数合并逻辑
- 修复 x64 交叉编译包内缺少 native binary 的问题

## [1.1.0] - 2026-05-15

### 新功能

- **Mermaid 图表渲染**：CodeBlock 支持 Mermaid 图表实时渲染，可在预览与源码间切换；推理内容中内嵌 Mermaid 不再触发渲染
- **HTML 文件预览**：HTML 文件改为 webview 渲染，支持真实页面预览而非展示源码
- **附件持久化**：附件文件现可跨会话持久化，会话视图同步改造
- **工作目录侧栏折叠**：WorkSidebar 工作目录分区支持折叠并记忆折叠状态
- **Markdown 预览**：CodeBlock 支持 Markdown 预览与原始内容切换
- **HATCH 孵化流程**：重写 HATCH 孵化流程，与 user-profile 完整联动
- **checkPathsExist 接口**：desktopApi 补充 `checkPathsExist` 方法桩

### 性能优化

- **消除启动白屏**：优化启动性能，消除应用启动时白屏闪烁
- **webview 预加载**：webview 在动画开始前提前加载，消除打开时的白屏

### 界面优化

- **设置窗口宽度限制**：打开设置时强制窗口宽度不低于 960px，关闭后恢复限制
- **设置页装饰图**：设置页装饰图随窗口宽度自适应缩放
- **首页提示文案**：优化首页提示文案样式与目录按钮展示
- **模型设置页面**：优化模型设置页面交互细节
- **主工作目录卡片**：重设计主工作目录设置卡片样式
- **侧栏交互**：优化侧栏交互体验

### Bug 修复

- **推理底部指示器**：修复推理中底部指示器抖动问题
