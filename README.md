# Finch（芬奇）

Finch 是一只停在你桌面上的小雀鸟。

她不会在你打开窗口时递上一堆按钮和菜单。她只是安静地待在那里，等你开口。你说一句话，她便飞出去，衔回线索；你丢给她一堆碎片，她便织成脉络。

## 她会做的事

**衔来线索，织成脉络**

你散落在各处的笔记、代码、文档、网页——Finch 会帮你串起来。她记得你三天前提过的那个函数名，也记得你上周随手记下的思路。问她"那个方案后来怎么样了"，她能接得上话。

**振翅连接，轻连各处**

Finch 能飞进你的工作区，在你真实的项目里翻找、阅读、整理。她不会要求你把所有东西搬到某个新平台里去。她就地取材，在你的代码库、本地文件和常用工具之间轻盈穿梭。

**守巢执行，稳步推进**

交给她一个复杂任务，她会一步一步推进，不催促，也不遗忘。你可以随时离开，回来问她"进行到哪了"，她会告诉你进度，接着做下去。

## 为什么是一只鸟

因为好的陪伴应该像鸟一样——不沉重，不喧嚣，该出现时落在手边，该忙碌时飞向远方。Finch 不用"功能列表"来定义自己，她只是一只愿意帮你整理世界的小雀鸟。
