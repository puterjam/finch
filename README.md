# Finch

Finch 是一款桌面 AI 智能体助手，希望能成为你的长期伙伴。

## 它是你的长期陪伴助手

Finch 不只是对话工具。她是有自己的个性，根据你的偏好来定义 —— 一个持续记忆的、身份和关系的个人空间。无需先选项目，随时开口提问。

当你需要深入具体工作时，Finch 可以进入任意项目的工作区，基于本地代码库和文档上下文作答、执行、整理。

## 核心能力

- **多轮智能对话** — 支持复杂任务拆解、工具调用和推理过程展示
- **工作区管理** — 在多项目之间自由切换，上下文精准隔离
- **会话历史** — 自动保存、随时恢复，长任务不怕中断
- **多模型接入** — 灵活配置国内外主流 AI 模型，自定义参数
- **丰富内容渲染** — Markdown 文档、Mermaid 图表、HTML 预览、代码高亮
- **附件持久化** — 上传的文件跨会话长期保存
- **内置技能生态** — 文档处理、表格分析、PPT 生成、前端设计、PDF 处理等即用型技能

## 适用场景

- 基于本地代码库进行 AI 辅助编程与问题排查
- 多项目并行的日常开发咨询
- 需要上下文延续的复杂长对话
- 文档、演示稿、数据表格的智能化处理
